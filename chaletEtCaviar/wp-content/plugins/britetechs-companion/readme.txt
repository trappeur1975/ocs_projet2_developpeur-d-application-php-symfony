Enhance britetechs WordPress Themes additional functionality.

== Description ==

This plugin is developed for britetechs themes. This plugin creates the additional sections on the home page in britetechs themes.
Britetechs Companion is a plugin creation to enhance the functionality of WordPress theme created by britetechs.
